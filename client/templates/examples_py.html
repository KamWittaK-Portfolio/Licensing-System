import requests URL = "http://localhost:5001/auth" LICENSE = "8APD1AJ80MIIT1DL" def auth(): data = { "license": LICENSE } try: response = requests.post(url=URL, json=data) response.raise_for_status() if response.status_code == 200: return "You are authenticated to use this code", True elif response.status_code == 201: return "Your time has run out", False else: return "Unexpected response. Ask the owner for access to use this code.", False except requests.exceptions.RequestException as e: return f"An error occurred: {e}", False def main(): authMessage, authStatus = auth() if not authStatus: print(authMessage) return # Rest of Code print("Hello World") if __name__ == "__main__": main()
